# ConferenceTicketGenerator
🎟️ A full-stack Conference Ticket Generator built with Flask that enables users to book event tickets, make payments via Razorpay, and receive QR-based PDF tickets via email. Includes user authentication, dashboard and admin panel.

🎟️ Conference Ticket Generator System

A full-stack web application that allows users to book, pay, and receive digital event tickets with QR codes and PDF format.

🚀 Features of the project
🔐 User Registration & Login
🎟️ Ticket Booking System
💳 Payment Integration using Razorpay
📱 QR Code Generation
📄 PDF Ticket Download
📧 Email Ticket Delivery (Flask-Mail)
📊 User Dashboard
👨‍💼 Admin Panel
🛠️ Tech Stack
Backend: Flask (Python)
Frontend: HTML, CSS, Bootstrap
Database: SQLite
Payment Gateway: Razorpay
Email Service: Gmail SMTP
🔄 Workflow
Register → Login → Book Ticket → Payment → 
QR + PDF Generated → Email Sent → Dashboard View
